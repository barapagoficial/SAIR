Política de Privacidad

Última actualización: 11/07/26

1. Recopilación de Información y API Keys
Para utilizar el chat de Inteligencia Artificial en el Sitio, el usuario debe introducir su propia API Key de Google AI Studio (Gemini).
Seguridad: Su API Key se procesa localmente en su propio navegador o se envía de forma directa y encriptada al proveedor del modelo de lenguaje (Google).
No Almacenamiento: Nosotros no almacenamos, no guardamos ni recopilamos sus API Keys en ninguna base de datos externa ni en servidores propios.

2. Cookies y Anuncios de Terceros (Google AdSense)
Este Sitio utiliza servicios de publicidad de terceros para mostrar anuncios y mantenerse financiado. Específicamente, utilizamos Google AdSense.
Google, como proveedor asociado, utiliza cookies para publicar anuncios en este Sitio.
El uso de la cookie de publicidad permite a Google y a sus socios mostrar anuncios basados en las visitas que los usuarios realizan en este Sitio y en otros sitios de Internet.
Los usuarios pueden inhabilitar el uso de la publicidad personalizada visitando la Configuración de anuncios de Google.

3. Herramientas de Análisis
Este Sitio puede utilizar herramientas básicas de análisis de tráfico para conocer el número de visitas e interacciones, con el único fin de mejorar la experiencia del usuario y el rendimiento de la aplicación. Estos datos son completamente anónimos.

4. Enlaces a Terceros
Nuestro Sitio contiene enlaces a plataformas externas (como Google AI Studio para la obtención de la API Key). No nos hacemos responsables de las políticas de privacidad ni de las prácticas de dichos sitios web externos.

5. Aceptación de los Términos
Al utilizar este Sitio, usted acepta los términos descritos en esta Política de Privacidad. Si no está de acuerdo con ellos, por favor, absténgase de utilizar el servicio.

6. Contacto
Si tiene alguna duda o pregunta sobre esta Política de Privacidad, puede contactarnos en: barapagoficial@gmail.com.

Al hacer clic en "Acepto" aceptas las políticas descritas.
